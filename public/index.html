Features

Everything you need,
nothing you don't

Built for agencies managing Cypress and Playwright tests across multiple clients and projects.

Cypress & Playwright Support

Run Cypress or Playwright test suites directly from the browser. Playwright projects are auto-discovered from your config — no CI pipeline required.

Real-Time Log Streaming

Watch test output appear line by line as it runs. Powered by WebSockets via Laravel Reverb — no polling, no page refreshes.

Branded Client Reports

Per-client logos, colours, and footer text. Each report is fully self-contained with inline screenshots, videos, and print-to-PDF support.

Secure Shareable Links

30-day HMAC-signed URLs let clients view reports without creating an account. No guessable paths, no open storage buckets.

Multi-Project & Branch Support

Map projects to separate Git repositories with deploy keys. Override branches per run and manage environment variables at the suite level.

Role-Based Access

Admin and PM roles keep your team's permissions appropriate. Admins configure; PMs trigger runs and review results.

How It Works

From zero to report
in three steps

Set it up once. Run it whenever you need. Share with confidence.

1

Configure

Add your project, connect your Git repository with a deploy key, define your test suites, and set your environment variables. Set up client branding once and it applies to every report.

project → Acme Corp
repo → git@github.com:acme/e2e.git
branch → main
suite → Smoke Tests
2

Run

Hit run from the dashboard. Your test suite is dispatched to a background queue and you watch the output stream live in your browser via WebSocket. No blind waiting.
